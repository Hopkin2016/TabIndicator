# TabIndicator
自定义View实现顶部Tab指示器，代码非常简单和简洁。另外Fragment切换时也能避免不必要重绘。

当然，你并不需要重复造轮子，官方Design库中已经提供TabLayout+ViewPager实现顶部Tab指示器，绘制什么的都在XML属性定义，而且提供了很多形式的Tab类型，功能极其强大。
